# Titanic Survival Analysis

## Overview

This project performs a basic analysis of the Titanic dataset using Python and Pandas. It demonstrates how to load data, inspect it, identify missing values, and calculate simple statistics.

## Features

- Load the Titanic dataset
- Display dataset information
- Show the first five rows
- Count missing values
- Calculate the survival rate
- Display passenger class distribution
- Display gender distribution

## Technologies Used

- Python
- Pandas

- `titanic_analysis.py` – Main Python program
